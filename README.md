# BookStore (Let's Read)
# It is a mobile app where a user can be both a seller and a buyer. This means a user can buy and sell books on the platform. It mainly focuses on students who love reading hardcover books but cannot always afford them. By using this app, they can buy newly published books and also sell their own books, helping them save money.
